Get bounds for passed functions. 

Algorithm of search\+:
\begin{DoxyItemize}
\item collect all symbols and their addresses (offsets) from modules, poited in functions-\/under-\/test info
\item sort all symbols with their offsets
\item for every function-\/under-\/test name searching for coincidence
\begin{DoxyItemize}
\item if there is not exact coincidence and {\ttfamily use\+\_\+pattern = true} will search pattern coincidence
\item if failure and {\ttfamily use\+\_\+default\+\_\+bounds = true} and default-\/address is correct will take default-\/address as function bounds
\item else will ask user to enter bounds manualy
\end{DoxyItemize}
\item return resulting dict with function bounds
\end{DoxyItemize}
